
# React-shop is a React application.

Frontend React app.

## Installation

Clone my project, after type in terminal:

```bash
  npm install 
  npm run dev
```

## Tech Stack

HTML, CSS, SCSS, React, React-router, Redux-toolkit, Axios, Aint-Design, Fontawesome.
